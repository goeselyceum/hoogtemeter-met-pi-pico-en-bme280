Datalogger met een rp2350, een BME180 en een BMI160 om de hoogte van een waterraket te meten.

Features:
1: kallibreren van de sensoren tijden de pré-launch fase. Led knippert rood
2: Lanceer detectie mbv bmi160, led knippert groen
3: vluchtfase led brandt blauw, apogeum detectie met bmi160 hoogte meting met bme280
4: landing detectie, vluchtdtat opslaan in een csv-bestand, led brandt continue groen
